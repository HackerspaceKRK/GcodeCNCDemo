This is a simple demonstration of a 2-axis CNC machine using an Arduino UNO and the Adafruit Motor Shield v1 and v2.

##Which files should I use?

If you are testing out X-Y table then use 2axis.ino .
If you already made some wood works and have also Z axis, then use 4axis.ino .

##Prerequisites

Arduino: http://arduino.cc/
A lot of hot glue
Some more hot glue
